Features

Everything you need,
nothing you don't.

Built for developers who want a fast, private API client that stays out of the way.

Request Builder

Full control over method, URL, headers, query params, body, and auth. Syntax-highlighted JSON editor included.

Collections

Organize requests into collections with nested folders. Keep your API workspace structured and navigable.

Environment Variables

Define variables per environment — dev, staging, production — and reference them with {{base_url}} syntax.

Git Sync

Push and pull collections to GitHub or GitLab. Share API definitions with your team through version control.

Vault

Store tokens, API keys, and secrets in an encrypted vault. Sensitive values never leak into collection files.

Desktop Native

Runs as a native desktop app via NativePHP. No browser tab needed — your data stays on your machine.
